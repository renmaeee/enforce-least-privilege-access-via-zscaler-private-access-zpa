# Work Breakdown Structure

## Milestones

### Week 1: Web Application Reconnaissance & Manual Vulnerability Assessment (Part 1)
- **Status:** todo
- **Priority:** medium
- **Due:** N/A

Students will initiate project setup, familiarize themselves with essential security tools, conduct initial reconnaissance on target websites, and begin manual web application vulnerability assessment using the OWASP Top 10 framework.

#### Tasks
- [ ] **Target Reconnaissance & Initial Mapping** (todo) — medium priority
- [ ] **Manual OWASP Top 10 Vulnerability Assessment (Phase 1)** (todo) — medium priority

### Week 2: Manual Vulnerability Assessment (Part 2) & Automated Scanning Initiation
- **Status:** todo
- **Priority:** medium
- **Due:** N/A

Students will continue manual web application vulnerability assessment, aiming to identify a minimum number of distinct vulnerabilities. They will then finalize manual documentation and set up Nessus for automated scanning.

#### Tasks
- [ ] **Manual OWASP Top 10 Vulnerability Assessment (Phase 2)** (todo) — medium priority
- [ ] **Vulnerability Documentation & Reporting** (todo) — medium priority
- [ ] **Automated Vulnerability Scanning Setup** (todo) — medium priority

### Week 3: Automated Vulnerability Analysis & ZPA Architectural Design (Part 1)
- **Status:** todo
- **Priority:** medium
- **Due:** N/A

Students will execute automated vulnerability scans using Nessus, analyze the results, and begin the foundational design of a Zscaler Private Access (ZPA) architecture for an institutional network.

#### Tasks
- [ ] **Execute & Analyze Automated Scans** (todo) — medium priority
- [ ] **Generate Nessus Scan Reports** (todo) — medium priority
- [ ] **ZPA Architectural Blueprint (Phase 1)** (todo) — medium priority

### Week 4: ZPA Architectural Design (Part 2), Use Cases & Final Deliverables
- **Status:** todo
- **Priority:** medium
- **Due:** N/A

Students will finalize the ZPA architectural design, detail specific access use cases, and compile all project deliverables for final submission and presentation.

#### Tasks
- [ ] **ZPA Architectural Blueprint (Phase 2)** (todo) — medium priority
- [ ] **Develop ZPA Use Case Scenarios** (todo) — medium priority
- [ ] **Synthesize & Present Findings** (todo) — medium priority
